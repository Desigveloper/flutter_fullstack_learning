# Your six big projects by tiers

The 6 projects are grouped into 3 tiers of 2 projects each, ordered by difficulty. Every tier introduces new skills that the next tier depends on.

## 🟢 TIER 1 — Foundation

Goal: Get comfortable building real screens, connecting UI to logic, and handling basic data.

## 🟡 TIER 2 — Intermediate

Goal: Build apps with real data complexity, multi-user logic, and patterns that scale.

## 🔴 TIER 3 — Advanced

Goal: Build production-grade, real-time apps with complex architecture. Portfolio-defining work.

📝 Pro Tip

Document each project in its own README.md:

- What you learned
- Tech stack used
- Challenges faced
- Future improvements

This way, your folder becomes not just a learning archive but also a portfolio you can show to employers or collaborators.
