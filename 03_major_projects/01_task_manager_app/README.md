# Project 1 — Task Manager App

The simplest complete app you can build. Master the fundamentals here so everything else is easier.

What it is: A personal task manager. Create tasks, mark them done, delete them, filter by status.
What you'll learn:

Local state management with setState
CRUD operations (Create, Read, Update, Delete) in its purest form
Lists, rendering dynamic data
Basic navigation (screens)
Form input and validation
Local data persistence with shared_preferences

Why this first: Every app you will ever build does CRUD. This project teaches it with zero complexity in the business logic — so you can focus entirely on the Flutter fundamentals.
