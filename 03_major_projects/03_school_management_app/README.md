# Project 3 — School Management Mini App

Real data relationships. One app, multiple roles, multiple data types.

What it is: A school admin tool. Teachers, students, classes, grades, attendance — all connected.
What you'll learn:

Data modeling (classes that represent real-world entities)
Relational data in Firestore (subcollections, references)
Multi-user roles with different permissions
State management with Provider (your first proper state solution)
Reusable widgets built from scratch

Why third: You know how to connect to Firebase. Now learn how to structure data properly — the skill that separates junior from mid-level engineers.
